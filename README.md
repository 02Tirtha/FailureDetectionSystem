# Failure Detector (Ongoing)

Silent Failure Detection System: Full stack application that detects missing or delayed workflow steps, logs failures, sends email alerts, and includes a scheduler for automatic monitoring.

---

## 🚀 Features
- Workflow monitoring
- Failure detection system
- Dashboard with basic stats
- Event testing module
- Email alerts (in progress)

---

## 🛠️ Tech Stack
- Backend: Spring Boot
- Frontend: React
- Database: MySQL
- Deployment: Docker (learning), Railway

---

## 📌 Status
🚧 Currently under development

---

## ▶️ How to Run

1. Clone the repository  
2. Configure MySQL in `application.properties`  
3. Run backend (Spring Boot)  
4. Run frontend (React)
